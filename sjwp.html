Što smo naučili?

HTML struktura

DOCTYPE deklaracija, semantički elementi kao što su nav, main, section i footer. Atributi, ugniježđivanje i organizacija dokumenta.

CSS oblikovanje

Selektori, box model, flexbox, tipografija i boje. Razlika između inline, internih i eksternih CSS datoteka.

Planiranje strukture sjedišta (site map), organizacija mapa i resursa, konzistentnost dizajna i hosting.
