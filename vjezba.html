Što je HTML forma?

HTML forma (<form>) je element koji korisniku omogućuje unos i slanje podataka. Elementima forme skupljamo razne tipove podataka, a JavaScript ih može obrađivati dinamički bez ponovnog učitavanja stranice.

  • <input type="text"> — tekstualni unos
  • <input type="number"> — numerički unos s min/max ograničenjima
  • <select> — padajući izbornik s opcijama
  • <datalist> — predložene opcije uz slobodan unos
  • <label> — oznaka koja opisuje element
  • <button type="submit"> — gumb koji pokreće obradu forme

HTML kod forme

<form id="anketa" onsubmit="handleSubmit(event)">

  <label for="ime">Ime i prezime</label>
  <input type="text" id="ime" required>

  <label for="razred">Razred</label>
  <select id="razred" required>
    <option value="" disabled selected>— odaberi —</option>
    <option>3. razred</option>
  </select>

  <label for="sati">Sati na društvenim mrežama dnevno</label>
  <input type="number" id="sati" min="0" max="24" required>

  <label for="platforma">Omiljena aplikacija</label>
  <input type="text" id="platforma" list="platforme" required>
  <datalist id="platforme">
    <option value="TikTok">
    <option value="Instagram">
  </datalist>

  <button type="submit">Pošalji odgovor</button>
</form>

Što je HTML tablica?

HTML tablica (<table>) prikazuje podatke u redovima i stupcima.

  • <thead> — zaglavlje tablice
  • <tbody> — tijelo tablice s podacima
  • <tr> — jedan redak
  • <th> — ćelija zaglavlja
  • <td> — ćelija s podatkom
<table>
  <thead>
    <tr><th>Ime</th><th>Razred</th><th>Sati</th></tr>
  </thead>
  <tbody>
    <tr>
      <td>Ana Horvat</td>
      <td>3. razred</td>
      <td>4</td>
    </tr>
  </tbody>
</table>
